# Portfolio Brutaliste

Ce dépôt contient le code source de mon portfolio personnel. Conçu avec une approche **brutaliste**, il met en avant mes projets en tant qu'étudiant en **BTS SIO - SISR** (Solutions d'Infrastructure, Systèmes et Réseaux).

## Concept & Design

Le design repose sur les principes du **web brutalism** :
- **Typographie forte** : Utilisation massive de polices Sans-Serif.
- **Contraste élevé** : Palette de couleurs réduite (Noir, Blanc, Greenyellow) pour rappeler les interfaces de terminaux et la documentation technique.
- **Réactivité totale** : Interface "Liquid Layout" utilisant les unités de viewport (`vw`, `cqw`) et `clamp()` pour maintenir l'intégrité du design sur tous les supports.

## Stack Technique

- **Frontend** : HTML5 / CSS3 (Grid & Flexbox) pur, sans framework pour une performance maximale.
- **Graphisme** : SVG Inline et motifs d'arrière-plan générés dynamiquement.

## Infrastructure & Déploiement

En tant qu'étudiant SISR, ce portfolio n'est pas seulement une vitrine, mais une partie intégrante d'un écosystème auto-hébergé :
- **Hébergement** : Auto-hébergé sur mon infrastructure personnelle.
- **Documentation** : Toutes les procédures techniques liées à cette infrastructure sont/seront centralisées sur mon instance [HWIKI.XYZ](https://hwiki.xyz).

## Installation locale

Si vous souhaitez explorer le code localement :

1. Cloner le dépôt :


2. Ouvrir `index.html` dans votre navigateur.
